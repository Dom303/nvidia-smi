# nvidia-smi

Shows the GPU usage at status bar.

![nvidia-smi GPU stats shown](images/nvidia-smi.png)

## Styles

* circle `◌◔◑◕◍`
* bar `▁▂▃▄▅▆▇█`
* recycle `♺♳♴♵♶♷♸♹`
* die `⛶⚀⚁⚂⚃⚄⚅`
* clock `🕛🕐🕑🕒🕓🕔🕕🕖🕗🕘🕙🕚`
* line `⎽⎼⎻⎺`
* pile `𝄖𝄗𝄘𝄙𝄚𝄛`
* digit `0123456789`
* circledigit `🄋➀➁➂➃➄➅➆➇➈`
* negativecircledigit `🄌➊➋➌➍➎➏➐➑➒`
* wan `🀆🀈🀉🀊🀋🀌🀍🀎🀏`
* tiao `🀆🀐🀑🀒🀓🀔🀕🀖🀗🀘`
* bing `🀆🀙🀚🀛🀜🀝🀞🀟🀠🀡`
